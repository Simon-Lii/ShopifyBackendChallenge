Open new terminal, install backend server

1) cd backend
2) virtualenv venv
3) ./venv/Scripts/activate (for windows)
3) source venv/bin/activate (for mac)
4) pip install -r requirements.txt
5) flask run

Open new terminal, install frontend server

1) cd frontend
2) npm install
3) npm start
